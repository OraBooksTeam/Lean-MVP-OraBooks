Webhook	Stripe, n8n, custom	Payment events, external sync	HMAC SHA256 + idempotency key	Dead letter queue, manual retry	1000/hour	10 failures → 120s open	10s	3 retries: 1m,5m,15m
Email	AWS SES / SendGrid	Notifications, verification	SMTP credentials, TLS	Backup provider, log file	200/24h (sandbox)	5 failures → 30s open	10s	3 retries: 1m,5m,15m
